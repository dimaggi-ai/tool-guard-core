## How this file is maintained

Regenerate after any `go.mod` change:

```sh
go list -m -json all | python3 -c '...'  # see scripts/regen-notices.sh
```

The GoReleaser pipeline (`.goreleaser.yaml`) also emits an SBOM via
syft alongside each release tarball. Combined, the two artifacts give
auditors both the human-readable license list AND the
machine-readable SBOM.
